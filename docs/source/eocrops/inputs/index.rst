Inputs
======

Using EOCrops, you can have access to several data sources:

* Satellite data : Sentinel-1, Sentinel-2, PlanetScope, SPOT and Pleaides
* Weather data : ERA5 and ERA5T using meteoblue API
* Copernicus products : SentinelHub hosts several data sources publicly available (land cover, copernicus products)
